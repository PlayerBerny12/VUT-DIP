## Framework pro detekci deepfakes

**Autor:** Bernard Jan, Bc. - xberna18@stud.fit.vutbr.cz \
**Vedoucí:** Malinka Kamil, Mgr., Ph.D. (UITS FIT VUT) - malinka@fit.vut.cz

---

**Zadání:**
1.  Seznamte se s problematikou deepfakes (hlas a video). Prozkoumejte současný stav metod detekce deepfakes (hlas a video).

2.  Seznamte se technologiemi potřebnými pro tvorbu webových rozšíření a technologiemi pro tvorbu škálovatelných serverových aplikací.

3.  Seznamte se s již existujícími řešeními detekce deepfakes (např. jiné komerční webové pluginy).

4.  Navrhněte rozšiřitelný framework (server-client nebo pouze klientský) pro detekci deepfakes (podpora min. 3 detekčních metod (hlas i video). Navrhněte webové rozšíření pro detekci deepfakes, který bude využívat tento framework. Řešení má podporovat více prohlížečů a umožnit detekci v zobrazovaném obsahu i v nahraných souborech.

5.  Implementujte nástroj podle zpracovaného návrhu.

6.  Proveďte testování funkčnosti a spolehlivosti výsledné implementace. Testování proveďte alespoň nad dvěma nezávislými veřejně dostuponými datasety deepfakes.

7.  Diskutujte použitelnost, efektivitu detekce a možná rozšíření.
